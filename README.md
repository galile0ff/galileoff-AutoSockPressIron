## âš™ï¸ How to Use

1. **Clone** the repository to your local machine.  
2. Check `/Docs` for **assembly instructions** and **wiring diagrams**.  
3. Upload **Arduino firmware** from `/Firmware` to your microcontroller.  
4. Assemble **mechanical parts** from `/Mechanics` and `/3D-Models`.

6. Start the conveyor and monitor ironing & printing operations.
